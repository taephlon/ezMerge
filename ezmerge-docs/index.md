# ezMerge Documentation

Welcome to the documentation for **ezMerge**, a modern suite of tools designed to make Gentoo overlays, dependency resolution, and package management beginner-friendly while preserving the raw power and flexibility of Gentoo Linux.

## 📦 What is ezMerge?

ezMerge is an ecosystem consisting of multiple components:

1. **ezmerge-cli**: The main command-line assistant (written in Rust) that simplifies installing packages from overlays.
2. **ezmerge-overlay**: A curated, high-quality Gentoo repository containing ebuilds that conform to strict quality guidelines.
3. **ezmerge-web**: An online, interactive packages and overlay search portal.
4. **ezmerge-api**: The web backend providing package indexing and overlay trust telemetry.
